# Student Performance Analysis System

## Overview

This project analyzes simulated student performance data using Python and Pandas. It is designed to demonstrate how data analytics can help educators identify academic patterns, at-risk students, and performance levels.

## Tools Used

- Python
- Pandas
- Matplotlib

## Features

- Calculates average student score
- Calculates average attendance
- Identifies at-risk students with scores below 60
- Groups students into High, Medium, and Low performance levels
- Generates a performance distribution chart

## Dataset

The dataset includes simulated classroom information:

- Student ID
- Score
- Attendance percentage
- Assignments completed

## Key Insight

Data analysis can help educators identify struggling students earlier and support more targeted academic intervention.
